Real-time 3D video textures
for GStreamer

fluxvideotex is a GStreamer element that uploads live video frames as GPU textures onto any GLB 3D scene, rendered offscreen via Filament with full HDR color-grading support.

View on GitHub Get started

What it does

Each incoming RGBA video frame is uploaded as a Filament texture, bound to the baseColorMap of every material in your GLB scene, rendered offscreen via OpenGL, and emitted downstream as a new video frame — all inside a standard GStreamer transform element.

🎬

Any GLB model

Load any glTF 2.0 binary asset at runtime via the glb-file property. Falls back to a built-in cube.

🌟

HDR color grading

sRGB, BT.709, Rec.709 Linear, Rec.2020 Linear, PQ (ST.2084), and HLG output color spaces.

🆕

Y'CbCr output

Optional YCbCr encoding via Filament ColorGrading. Emits AYUV on the src pad.

⚙

GStreamer native

Drop-in BaseTransform element. Plug anywhere in a pipeline between videoconvert elements.

Build & install

Prerequisites

macOS (Apple Silicon), GStreamer 1.22+, CMake 3.20+, a local build of jesusluque/filament.

Clone and build

git clone https://github.com/jesusluque/flux
cd flux/tools/filament
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --target gstfluxvideotex

Register the plugin

export GST_PLUGIN_PATH=/path/to/flux/tools/filament/build/gst-fluxvideotex

# Verify
gst-inspect-1.0 fluxvideotex

Usage

Built-in cube (quickstart)

gst-launch-1.0 \
  videotestsrc pattern=smpte is-live=true \
  ! videoconvert \
  ! "video/x-raw,format=RGBA,width=1280,height=720,framerate=30/1" \
  ! fluxvideotex width=1280 height=720 \
  ! "video/x-raw,format=RGBA,width=1280,height=720" \
  ! videoconvert ! glimagesink

Custom GLB model

gst-launch-1.0 \
  videotestsrc pattern=smpte is-live=true \
  ! videoconvert \
  ! "video/x-raw,format=RGBA,width=1280,height=720,framerate=30/1" \
  ! fluxvideotex glb-file=/path/to/model.glb width=1280 height=720 \
  ! "video/x-raw,format=RGBA,width=1280,height=720" \
  ! videoconvert ! glimagesink

HDR output (Rec.2020 PQ)

! fluxvideotex glb-file=model.glb color-space=rec2020-pq width=1280 height=720 !

Y'CbCr output

! fluxvideotex glb-file=model.glb ycbcr-output=true width=1280 height=720 \
! "video/x-raw,format=AYUV,width=1280,height=720" !

poc003 demo app

# Built-in cube
GST_PLUGIN_PATH=.../gst-fluxvideotex \
open -a /tmp/poc003.app --args --duration 60

# Custom model with HDR
GST_PLUGIN_PATH=.../gst-fluxvideotex \
open -a /tmp/poc003.app --args \
  --glb /path/to/model.glb \
  --color-space rec2020-pq \
  --duration 60

Element properties

Property Type Default Description
glb-file string unset Path to a GLB asset. When unset the built-in cube is used.
width uint 1280 Rendered output width in pixels.
height uint 720 Rendered output height in pixels.
color-space enum srgb srgb · bt709 · rec709-linear · rec2020-linear · rec2020-pq · rec2020-hlg
ycbcr-output boolean false Encode output as Y'CbCr. Src pad emits AYUV instead of RGBA.
rotation-period-x double 150 Seconds per full X-axis rotation.
rotation-period-y double 200 Seconds per full Y-axis rotation.
rotation-period-z double 300 Seconds per full Z-axis rotation.
